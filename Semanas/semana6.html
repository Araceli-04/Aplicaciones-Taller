Semana 6

En esta semana se debe realizar la conexión del proyecto “Psicocitas” entre Java (NetBeans) y la base de datos MySQL administrada mediante XAMPP.

El propósito de esta actividad es establecer una conexión funcional y estable entre la aplicación y la base de datos, permitiendo ejecutar correctamente las operaciones de registro, consulta, actualización y eliminación de información desde la interfaz desarrollada en NetBeans.
