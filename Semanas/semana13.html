IMPLEMENTACIÓN DE LA BASE DE DATOS
Se ha diseñado e implementado la base de datos del sistema de Gestión de Tesis, permitiendo el almacenamiento, organización y recuperación eficiente de la información.

La base de datos integra las principales entidades del sistema, como usuarios, tesis y evaluaciones, asegurando la integridad de los datos y la correcta relación entre los diferentes componentes de la aplicación.

Además, se ha establecido la conexión entre la aplicación y la base de datos, garantizando la persistencia de la información y el correcto funcionamiento del sistema.
