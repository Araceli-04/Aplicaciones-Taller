Semana 4

En esta semana realizarás dos actividades principales:

1️⃣ Primera actividad: Modifica el último archivo o ejemplo trabajado para que el programa pueda calcular el área y el perímetro de tres figuras geométricas. Además, agrega una imagen representativa de cada figura.

2️⃣ Segunda actividad: Crea una aplicación que contenga una ventana de Login (con uno o dos roles), que valide usuarios con diferentes permisos, como alumno o docente. Luego, desarrolla una segunda ventana para realizar conversiones de unidades de medida (metro, centímetro, milímetro, etc.). Cada rol debe tener su propia vista: el estudiante solo podrá ver las conversiones, mientras que el docente podrá modificarlas o personalizarlas.
